Nexus 5
=======
**How to flash**:

- Reboot device to fastboot
- `fastboot flash imgdata imgdata.img`

Fairphone 2
===========
**How to flash**:

- Reboot device to fastboot
- In a terminal and do:
- `fastboot flash splash splash.img`
